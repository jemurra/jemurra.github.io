Leaflet Add Layer based on Div id

Trying to create a function that will add a map layer to leaflet based on the id of the checkbox div. Using 'this.id' would not translate to leaflet as a valid variable for a map layer of the same name. For example, loggin the checkbox id would result in a string of, say, 'Mountains'. Say I had a geojson marker layer in leaftet called 'Mountains'. Passing an event listner to a checkbox such as 'map.addLayer($(this).attr('id'));' would result in an error of 'TypeError' in leaflet js. What needed to happen is, add window object so the browser can communicate with the current window objects. So in the end: 'map.addLayer(window[$(this).attr('id')]);' = success!
Yeah.
I know.

Random Characters

Simple to use js function to create a string of x number of random characters from an array. Uses math.random to generate a random number and times this by the length of the array. Then uses math.floor to round down that number and pulls the character from the array at the number. Does this for X amount of times for how long you want your random string.
